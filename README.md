# Odin todoist

A simple `Todoist clone` 

## Built 

- JavaScript
- Webpack (bundled)
- date-fns (libraries)
- Local Storage ()
- CSS
- HTML

## Features

- Add, edit, and delete tasks
- Mark tasks as completed
- Organize tasks by projects and due dates
- Responsive design
